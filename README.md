## What it does

Minecraft Utilities packages a collection of web-based helpers for players who want a more organized way to plan and manage their Minecraft sessions. From laying out a large redstone build to working out the materials needed for a major project, or keeping tabs on exploration and progress, the suite gives you a straightforward place to work with your notes and numbers.

The guiding idea is simple: keep planning separate from play. Rather than relying on spreadsheets or outside apps, you get a dedicated browser environment for testing block totals, reviewing material ratios, and saving setups for later. Everything runs on the client side, so your data stays on your device unless you decide to export it.

## Features

- **Build Planner** - Estimate block quantities and material costs for structures of any size
- **Resource Tracker** - Log collected items and compare against your crafting goals
- **Enchantment Calculator** - Determine optimal enchantment combinations and anvil costs
- **Brewing Guide** - Reference potion recipes with ingredient dependency trees
- **Seed Map Viewer** - Visualize world generation parameters from your seed values
- **Inventory Organizer** - Simulate chest sorting and item grouping for storage rooms
- **Progress Dashboard** - Track achievements, advancements, and biome exploration status
- **Export/Import** - Save your tool configurations as JSON files for later use

## Getting started

There is nothing to install. Minecraft Utilities is designed to run entirely in the browser.

1. Download the latest build from the link above
2. Extract the ZIP archive to any folder on your computer
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, or Safari)
4. Start using the tools immediately - no server or internet connection needed after first load

For offline use, keep the folder available locally. Your information is stored in your browser's local storage.

## Settings and shortcuts

Every tool exposes configurable options. The table below summarizes the common controls:

| Setting | Default | Description |
|---------|-------------|-------------|
| Block rounding | On | Rounds material counts to nearest stack size |
| Auto-save interval | 5 min | Saves your current session data automatically |
| Dark mode | Off | Toggles between light and dark color themes |
| Measurement unit | Metric | Switches between blocks and meters |
| Export format | JSON | Choose between JSON, CSV, or plain text exports |

Keyboard shortcuts are available for faster movement through the app:

- `Ctrl+N` — New project
- `Ctrl+S` — Save current session
- `Ctrl+E` — Export data
- `Ctrl+I` — Import data

## Common questions

**How do I begin?**  
Open `index.html` in your browser. From the main menu, choose any available tool to open its interface.

**Will my data persist after I close the browser?**  
Yes. Configurations, project data, and preferences are stored in your browser's local storage. If you clear browser cache data, those records will be removed.

**Can I change how the tools behave?**  
Yes. Each tool includes its own settings area for calculation behavior, visual preferences, and export choices. No coding is needed.

**Is it usable on phones or tablets?**  
The layout is responsive and works on mobile devices, but some heavier tools like the seed map viewer are easier to use on a larger display.

**Where is the data kept?**  
Everything remains in your browser's local storage. Nothing is transmitted to a server. You can also export your data to keep a backup copy.

**How do I move to a newer release?**  
Download the latest build and replace the existing folder with the updated files. Your saved data should remain available as long as you keep using the same browser.
